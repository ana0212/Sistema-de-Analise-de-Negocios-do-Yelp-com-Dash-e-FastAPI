# Notebooks – Análise e Modelagem

Esta pasta contém os notebooks utilizados no desenvolvimento do projeto, abrangendo desde a análise exploratória até o treinamento dos modelos de Machine Learning.

## Organização

Os notebooks seguem uma abordagem sequencial:

- **Análise Exploratória**  
  Análise Exploratória de Dados (EDA) e na Análise Inferencial, contemplando desde a preparação e limpeza inicial dos dados até a investigação estatística das relações entre variáveis relevantes do negócio.

- **Análise de Cluster**  
  Identificação grupos distintos de estabelecimentos com base em avaliação média e volume de engajamento, caracterização do perfil de cada cluster e discussão de insights que possam apoiar a tomada de decisão.

- **Análise de Sentimento (Modelagem de classificação)**  
  - Preparação e transformação dos textos das avaliações para modelagem supervisionada, treinamento e avaliação de modelos de classificação.

## Observação Importante

Devido ao tamanho dos datasets originais do Yelp, os arquivos completos não estão versionados no repositório.  
Os notebooks assumem a existência dos dados localmente para fins de reprodução.

## Objetivo

Os notebooks documentam todo o raciocínio analítico e as decisões de modelagem, servindo como base técnica para os modelos utilizados no backend e no dashboard.
